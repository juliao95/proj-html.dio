O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criação de páginas da web. Ele estrutura o conteúdo da página usando elementos, também chamados de "tags".

HTML5 é a versão mais recente do HTML e traz melhorias em semântica, acessibilidade e suporte multimídia.

Índice Explicativo das Tags

  1. <br> — Insere uma quebra de linha no texto.

    Exemplo: "Linha 1<br>Linha 2" exibirá:
    Linha 1
    Linha 2

  2. <mark> — Destaca uma parte do texto, geralmente com fundo amarelo.

    Exemplo: "Isso é um texto marcado."

  3. <ol> — Cria uma lista ordenada (numerada).

    Exemplo:

    1. Primeiro item
    2. Segundo item
  4. <ul> — Cria uma lista não ordenada (com marcadores).

    Exemplo:

    • Item A
    • Item B
  5. <main> — Define o conteúdo principal da página. Deve conter a parte mais importante.

    É uma tag semântica usada apenas uma vez por página.

  6. <footer> — Representa o rodapé da página ou seção. Pode conter créditos, links ou informações legais.

  7. <header> — Representa o cabeçalho da página ou seção. Pode incluir logotipo, menu, título, etc.
